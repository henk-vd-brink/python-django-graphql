## About the Project
This project contains a template for setting up a GraphQL API using the Django framework. In addition, a database is spun up together with a reverse proxy (nginx).

## Getting Started

### Prerequistes
- Docker

### Installation
```
git clone https://github.com/henk-vd-brink/python-django-graphql.git
cd python-django-graphql
```

### Run
```
docker-compose up --build
```
